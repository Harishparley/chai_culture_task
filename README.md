## üé® Design & Features

### 1. The "Royal" Aesthetic
* **Color Palette:** Strictly adhered to the brand's requirement of Deep Brown (`#3E2723`) and Royal Gold (`#D4AF37`) to evoke warmth and luxury.
* **Typography:** Used *Playfair Display* for headings to reflect heritage and tradition, paired with *Lato* for modern readability.

### 2. Technical Highlights
* **Vertical Gradient Overlay:** Implemented a custom CSS `linear-gradient` on the hero section. It transitions from solid dark brown (top 50%) to transparent (bottom), ensuring text readability while smoothly revealing the high-quality tea background image at the footer.
* **Responsive Grid Layout:** The "Royal Flavors" section uses a flexible flexbox layout that displays 5 cards in a single row on desktop and stacks them elegantly on mobile devices.
* **Glassmorphism & Hover Effects:** Flavor cards feature a subtle lift animation (`transform: translateY`) and a dark glass-like background on hover to encourage user interaction.
* **Full-Width Card Imagery:** Product cards are designed with `object-fit: cover` to maximize visual impact without distorting the product images.

## üöÄ Getting Started

Follow these instructions to run the project locally on your machine.

### Prerequisites
Make sure you have **Node.js** installed.

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Harishparley/chai_culture_task
    cd chai-culture-landing
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm start
    ```

4.  Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
